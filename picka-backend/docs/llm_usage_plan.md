
# LLM 사용 계획

## LLM을 사용하지 않는 기능

- 할인 금액 계산
- 적립 금액 계산
- 전월 실적 확인
- 건당 한도 적용
- 월 한도 적용
- 카드 순위 계산
- 500원 이하 차이 타이브레이커

## LLM을 사용할 수 있는 기능

- 구조화되지 않은 혜택 조건 해석
- 가맹점 업종이 불명확한 경우 판단
- 혜택 제외 조건 해석
- 사용자에게 보여줄 추천 이유 생성

## LLM 입력 데이터

- 사용자의 보유 카드 중 관련 카드만 전달
- 관련 혜택 조건만 전달
- 가맹점명
- 결제금액
- Python 계산 결과

## LLM에 보내지 않을 데이터

- 전체 카드 1,558개
- 전체 사용자의 결제 기록
- 불필요한 개인정보
- 카드번호 전체
