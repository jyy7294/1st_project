
# 추천 로직 사용 필드

## recommend_cards 함수

입력값:

- 사용자 카드 목록
- 가맹점명
- 결제 업종
- 결제금액

사용 필드:

- 카드명
- 전월 사용금액
- 혜택 목록

출력값:

- 추천 카드
- 예상 혜택
- 순위
- 추천 이유
- 다른 카드 목록

## calculate_card_benefit 함수

사용 필드:

- 혜택유형
- 혜택단위
- 혜택값
- 카테고리
- 전월 실적 조건
- 건당 한도
- 월 한도

## 현재 미구현 또는 확인 필요

- 월 혜택 사용량
- 월 이용 횟수
- 일 이용 횟수
- 카드 통합 한도
- 최소 결제금액
- 간편결제 조건
- 특정 요일·시간대 조건
